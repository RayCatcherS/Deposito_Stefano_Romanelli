# differenza tra git e github
- git è un sistema di controllo versione, github è una piattaforma online che ospita repository git
- git è uno strumento da linea di comando, github offre un'interfaccia web e desktop per gestire i repository

### è possibile branchare e committare con git?
- sì, git consente di creare branch e committare modifiche localmente, senza bisogno di github
- le pull request su github servono per proporre modifiche al repository principale, ma non sono obbligatorie per lavorare con git

# modulo
è un file che contiene codice Python, che può essere importato e utilizzato in altri file Python. I moduli consentono di organizzare il codice in modo più efficiente e riutilizzabile.

- ogni modulo deve avere una sua logica e funzionalità specifica (unitaria)
- un modulo gestisce un obiettivo specifico, non deve essere troppo generico o troppo complesso (esempio creare i pacchi)
- si importano con import

suddividere nelle parti necessarie che permettono la gestione

## OOP
- Un linguaggio di programmazione: **condizioni**, **iterazioni**, **esecuzione**
- Tipi di linguaggi:
  - **Interpretato** (lento ma più sicuro in esecuzione: quanto sono sicuro che funzioni sempre)
  - **Compilato**
  - **Eseguiti** (JavaScript) → più veloce o meno veloce

### Regole / concetti (OOP e principi)
- Incapsulamento
- Polimorfismo
- Ereditarietà
- Astrazione (principio dei linguaggi di programmazione)

## Python: note varie
- File Python con **lettera maiuscola**
- Python è **dinamico** e **non è fortemente tipizzato** (*dinamicamente forte*)
  - È possibile fare tipizzazione dinamica
  - Ma non è completamente dinamico perché una variabile dichiarata non può non avere il tipo
- È semplice e leggibile, usato per vari tipi di sviluppi (server, web)
- **Multiparadigma**, usato per programmazione:
  - Imperativa
  - Funzionale
- È interpretato (*pseudo compilato*)
- **Indentazione**: è bloccante → distanza tra codice e margine

## IDE
Un IDE è un software che raggruppa tutti gli strumenti essenziali in un’unica interfaccia:
- editor di codice
- debugging
- build

Note interfaccia:
- Barra di lato: operiamo sull’intero ecosistema
- Barra in alto (tool software): possiamo gestire l’intero ambiente
